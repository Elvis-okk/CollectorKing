使用说明
首页收集
选择模板开始收集证件照片。点击空白区域拍照，长按空白区域可从相册多选照片。
照片预览
点击已有照片可全屏预览，双指缩放，点击旋转按钮可旋转照片。
打包分享
收集完成后点击打包，所有照片将压缩为ZIP文件，可分享给他人。支持按证件类型或拍摄顺序命名，可按类型分文件夹或不分类打包。
照片压缩
拍照后自动压缩照片，可在设置中调节压缩质量，质量越低文件越小。
模板管理
在模板页创建、编辑模板。支持导入导出模板，导出为TXT文件，导入时自动恢复自定义证件类型。
历史记录
打包记录自动保存，可查看历史打包详情、继续编辑或重新分享。
关于
收集王
版本 1.4.4

收集王是一款证件材料收集助手，帮助您快速收集、整理和打包各类证件照片。支持自定义证件类型、模板管理、照片压缩、一键打包分享，让材料收集更高效便捷。
